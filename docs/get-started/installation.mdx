Todo-CLI is built with Python and can be installed in just a few steps using [**uv**](https://github.com/astral-sh/uv), a fast dependency manager.

---

### Requirements

- Python 3.11+
- [uv](https://github.com/astral-sh/uv) installed (`pip install uv` or see [installation instructions](https://github.com/astral-sh/uv#installation))
- Git (to clone the repository)

---

### Quick Setup

Clone the project:

```bash
git clone https://github.com/vidjinnangni/todo-cli.git
cd todo-cli
