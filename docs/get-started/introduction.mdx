## Todo CLI X

**A minimalist and efficient todo manager you can use right from your terminal.**

Todo CLI X is designed to be simple, fast, and distraction-free.\
No accounts, no sync, no bloated UI — just a focused command-line interface to manage your daily tasks.

It’s ideal for developers, terminal users, and anyone who prefers text-based productivity.

## Why Todo CLI X?

Todo CLI X is a simple and minimalist command-line tool to manage your daily tasks directly from the terminal.

- **Minimalist**: Just what you need — nothing more
- **Efficient**: Fast commands, keyboard-only workflow
- **Offline-first**: Tasks are stored locally in a JSON file
- **Built for the terminal**: No mouse, no browser, no distractions
- **Open source**: Read, modify, and contribute on GitHub

Ideal for developers, hackers, and anyone who prefers lightweight tools.
